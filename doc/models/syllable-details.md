
# Syllable Details

This custom type contains the syllable details for word API.

## Structure

`SyllableDetails`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `count` | `Integer` | Optional | The count of syllables. |
| `list` | `Array<String>` | Optional | The list of syllables. |

## Example (as JSON)

```json
{
  "count": 1,
  "list": [
    "wind"
  ]
}
```
